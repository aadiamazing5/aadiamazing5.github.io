Giorno

First Robotics Scouting App

Using JAVA and Android Studio, I created and implemented a scouting app for my FIRST Robotics team (Merge Robotics Team 2706). The app allows the user to input various statistics for every team participating in the competition, then creates a ‘pick-list’ that ranks the other teams so that we can decide which teams to ally with, and which teams to avoid. I worked with 2 other people for this app, allowing me to further improve my collaborative skills and proficiency with git and github. The source code is private, but screenshots from the app can be found here.

FIRST Robotics

Robot for 2020 'Infinte Recharge' Competition

This year, for the Infinite Recharge FIRST competition, my team (Merge Robotics Team 2706) has created our robot named “High Ground”. As part of the software team, I used JAVA and FIRST’s WPILib frameworks to design, program and test the robots ball intake system. The robot was made in collaboration with the whole team, so the implementation helped improve my collaboration skills and proficiency with git and github. Pictures and videos of the robot in action can be found on the team’s Instagram (@mergerobotics2706)
