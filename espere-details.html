Espere

  • Category: Web App

  • Project date: June 2020

  • Github: github.com/aadiamazing5/SetHacks

  • Demo Video: www.youtube.com/watch?v=sdIa_Wr_33g

The 2020 Set.Hacks() Hackathon required its participants to make a project that would target one (or more) of the United Nations Sustainable Development Goals. My team and I made Espere - a web application that allows users to search for Homeless Shelters, Food Banks and Clothing Donation Centres near them. Users can specify names, areas and methods of transport. Then, the application uses a map interface to show nearby services. Espere won first place, as can be seen in the fourth image. Espere was made using a Flask backend that served a Vue frontend. I worked on the backend with python and Flask to integrate the Foursqaure API and Google Maps API into the app.
