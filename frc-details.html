FRC Robot - High Ground

  • Category: Robot

  • Project date: September 2019 - March 2020

  • Project URL: github.com/FRC2706/2020-Robot-Code

With my team, Merge Robotics Team 2706, we participated in First Robotics 2019-2020 challenge, Infinite Recharge. To tackle this challenge, the team began work on our robot, named "High Ground". I was assigned to work on the intake system, which needed to take in balls, and transport them to a container. Along with a team of 4, I used JAVA and FIRST’s WPILib frameworks to design, program and test the robot. The robot was made in collaboration with the whole team, so the implementation helped improve my collaboration skills, my git and github skills, and my ability to apply code to real life machines. Pictures of the robot in action can be found on the team’s Instagram.
