Synergyy Website

  • Category: Website

  • Project date: July 2020

  • Github: github.com/SynergyyBot/Synergyy-Website

  • Project URL: www.synergyy.ml/

This website is used as an addition to the Synergyy Discord Bot project. It gives the user info on how to use the bot, how to add the bot to their server, and the features the bot has. The bot has a features page, an FAQ section, a reviews section and a contact form. The web development tools used to build this website include Bootstrap, SCSS, Jquery, Animate.css, Wow.js, PHP forms, and more! This great project helped me develop my web development and graphic design skills.
