Synergyy Bot

  • Category: Discord Bot

  • Project date: July 2020

  • Github: github.com/SynergyyBot/SynergyyBot

  • Project URL: www.synergyy.ml

Due to the COVID-19 Pandemic, many universities have decided to move all courses online for Fall 2020. This news inspired a friend and I to create Synergyy - a Discord Bot for students. The bot can be used on Discord (a popular messaging and voice calling application) by anyone. The bot features meeting creation, RSVP functionality, reminders, todo list creation, Google Calendar integration and more! The bot was created using Discord's python framework along with PostgreSQL to store user data. We hosted the bot on Heroku to ensure 24/7 access to the bot. We also made a website to advertise the bot and help new users.
